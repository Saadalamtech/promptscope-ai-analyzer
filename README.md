# PromptScope-ai-analyzer
PromptScope — AI Prompt Analyzer

PromptScope is a web-based application designed to analyze and improve user prompts by evaluating clarity, specificity, and ambiguity using rule-based logic.

Project Overview

The application simulates how AI systems evaluate prompt quality. It provides structured feedback, highlights vague or unclear input, and suggests improved versions of prompts.

Features

- Analyze prompts based on clarity, specificity, and ambiguity
- Highlight vague words and unclear phrases
- Generate improved versions of prompts
- Store analysis history using localStorage
- Simple authentication system (login/signup)
- Responsive and clean user interface

Core Functionality

The system uses rule-based logic to:

- Identify vague terms (e.g., "something", "etc")
- Evaluate sentence structure and clarity
- Detect missing context or ambiguity
- Assign scores for different evaluation metrics

Tech Stack

- HTML
- CSS
- JavaScript (Vanilla JS)
- localStorage (data persistence)

UI Structure

- Landing page with introduction
- Analyzer dashboard for prompt evaluation
- History page for previous prompts
- Authentication pages (Login/Signup)

- Understood prompt quality and AI interaction principles
- Improved application architecture and state handling
